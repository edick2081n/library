Тестовое задание - создание библиотеки
============================================================================

подготовка к запуску проекта
-----------------------------------------------------------------------------
Создание виртуального окружения проекта
> python -m venv env


Установка необходимых пакетов python
> pip install -r requirements.txt

Запуск проекта
------------------------------------------------------------------------------

> python manage.py runserver 80
